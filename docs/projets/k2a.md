# 📘 Projet K2A

## 🧩 Contexte

Le projet **K2A** est une application web de type e-commerce spécialisée dans la vente de filtres automobiles.  
Il a été développé dans le but de digitaliser la gestion des produits et des commandes.

---

## 🎯 Objectifs

Le projet a pour objectifs :

- Gérer les produits (ajout, modification, suppression)  
- Gérer les commandes des clients  
- Afficher les produits par catégories  
- Gérer le stock des produits  
- Mettre en place un CMS pour faciliter l’administration  

---

## ⚙️ Technologies utilisées

- PHP (natif)  
- MySQL  
- Bootstrap  
- JavaScript  

---

## 📁 Structure du projet

Le projet est organisé comme suit :

- `admin/` : gestion des produits et administration  
- `controller/` : logique métier  
- `component/` : composants réutilisables  
- `css/` : styles  
- `js/` : scripts  
- `images/` : ressources visuelles  
- `database/` : fichiers SQL  

---

## ⚙️ Fonctionnalités

### 🔹 Gestion des produits (CMS)

- Ajouter un produit  
- Modifier un produit  
- Supprimer un produit  

👉 Cette gestion est réalisée via un CMS permettant à l’administrateur de gérer les produits sans modifier le code.

---

### 🔹 Catalogue

- Affichage des produits  
- Filtrage par catégories  
- Consultation des détails  

---

### 🔹 Commandes

- Ajout au panier  
- Consultation du panier  
- Validation de commande
