\# 🚗 K2A Filtration



\## 📌 Description

K2A Filtration est une plateforme e-commerce dédiée à la gestion et à la vente de filtres automobiles (huile, air, carburant, etc.).  

Le système permet de présenter les produits aux utilisateurs et de gérer un catalogue via un espace administrateur (CMS).



\---



\## 🛠️ Technologies utilisées

\- PHP

\- MySQL

\- HTML / CSS

\- Bootstrap

\- JavaScript

\- Git \& GitHub



\---



\## ⚙️ Fonctionnalités



\### 👨‍💼 Administration (CMS)

\- Ajouter un produit

\- Modifier un produit

\- Supprimer un produit

\- Gérer les images des produits

\- Gérer les catégories



\### 🛍️ Côté utilisateur

\- Consulter les produits

\- Voir les détails d’un produit

\- Navigation par catégorie



\---



\## 🗂️ Structure du projet

\- `/images` : stockage des images produits

\- `/includes` : fichiers de configuration (connexion BD)

\- `/pages` : pages principales

\- `/admin` : espace d’administration (CMS)
